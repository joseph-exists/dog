Dog Stack Architecture

Full-stack Docker Compose — FastAPI · TimescaleDB · Traefik · Microservices

Core Services

  • • Frontend: React + TS + Vite + ShadCN + TanStack
  • • Backend: FastAPI (Python 3.10) with 4 workers
  • • Package management: uv (astral.sh)
  • • Traefik v3.6.7 reverse proxy with Let's Encrypt

Microservices

  • • Kennel: LXC container manager with GPU passthrough
  • • Tesser: SVG factory with core/export/bridge workers
  • • MCP MVP: Model Context Protocol server
  • • Gittin: Gogs git server for shadow versioning
  • • Outbox Worker: shadow repo background sync

Data Layer

  • • TimescaleDB (PostgreSQL 17) — primary database
  • • Redis 8.2 — message bus & caching layer
  • • Redis Pub/Sub channels: kennel:events, tesser:requests, tesser:responses
  • • Prestart service runs Alembic migrations on deploy

Dog Stack • docker-compose.yml • 11 services across 2 Docker networks
