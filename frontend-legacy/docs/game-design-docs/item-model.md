Version 2: Items:

Items are objects that can be used in a Story, are collected in a Persona's Inventory, and are collected in a Player's Inventory.

A Persona will keep track of:
  The Items they currently have
  The Items they have had before
  How Items have been used
  The impact that using an Item had (the Item's effect)
  How it was possible to use an item

Items can trigger Events and be effected by Events.
Items can have associated Traits and Qualities.
Items can have Temporary Effects on a Persona.
Items can have Permanent Effects on a Persona.

Items can have qualifying conditions before their Effects or Events are triggered.
